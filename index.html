Código: TTCT0003 | I Cuatrimestre 2026 | Modalidad híbrida

Acerca del curso

Curso teórico-práctico de 15 semanas diseñado para brindar al estudiante las herramientas matemáticas fundamentales del cálculo diferencial e integral, en el contexto del análisis de datos y la inteligencia artificial. Se exploran funciones, límites, derivadas e integrales en una y varias variables, junto con sus aplicaciones contemporáneas: optimización de funciones de costo mediante descenso de gradiente, diferenciación automática como base de las redes neuronales, transformadas de Fourier para análisis de imágenes y audio, y aproximaciones numéricas.

Herramientas y librerías principales

Google Colab

Plataforma principal para programación sin instalación. Incluye Python 3.12+ y GPUs gratuitas.

NumPy

Cálculo numérico, operaciones vectorizadas y álgebra lineal para implementar algoritmos desde cero.

SymPy

Matemática simbólica para resolver límites, calcular derivadas e integrales exactas, y manipulación algebraica.

SciPy

Integración numérica, optimización y métodos científicos avanzados para problemas reales.

Matplotlib

Visualización científica de datos. Creación de gráficos 2D y 3D para análisis matemático.

PyTorch

Diferenciación automática con torch.autograd y base para redes neuronales modernas.
